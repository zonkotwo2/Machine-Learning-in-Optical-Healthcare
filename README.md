# Human Brain Sensing Using Photonics

This repository contains my undergraduate thesis on brain-state classification using laser speckle imaging data.

## Summary
This project explored machine learning approaches for classifying brain states from laser speckle imaging data using Python and MATLAB.

## Contents
- Thesis.pdf
- figures/
- code/

## Methods used
- SVM
- Random Forest
- CNNs (VGG19, ResNet-18)
- Optical flow
- Contrast and cross-correlation processing
